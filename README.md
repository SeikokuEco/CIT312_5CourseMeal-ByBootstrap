# CIT312_5CourseMeal-ByBootstrap
5 Course Meal, By Bootstrap Assignment for CIT312


# Justification for Bootstrap 
     
  `col-md-6`: this was used for medium (desktop/tablets) so that the recipe appears in two columns
  
  `col-12`: used for mobile devices. realigns everything to be visually, in one column for easier reading
  
  `img-fluid`: used to dynamically fit images within containers so they don't overflow


  `display-4`: used for headings using bootstrap
  
  `text-center`: used to align text in the center using bootstrap


  `d-none d-md-block`: md and above shows header and hides h1
  
  `d-md-none`: breakpoints below md hides header
  - These were used so that the heading wasn't so gigantic on smaller screens
